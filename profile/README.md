## 4. 🔄 CfoE — Audit Flow

```mermaid
sequenceDiagram
    participant A as 👤 Analyst
    participant F as 🖥️ Frontend
    participant B as ⚙️ Backend API
    participant C as 🎯 Coordinator
    participant S as 📐 Scoring Engine
    participant P as 🛡️ Policy Engine
    participant G as 🤖 Groq LLM
    participant BC as ⛓️ Algorand
    participant H as 📦 History Store

    A->>F: Submit supplier data
    F->>B: POST /api/audit
    B->>C: Build audit workflow
    C->>S: calculate_carbon_score()
    S-->>C: risk_score + classification
    C->>P: enforce_policy_hitl(risk_score)
    P-->>C: decision + human_approval_required
    C->>G: Generate executive report
    alt Model available
        G-->>C: AI report text
    else Model unavailable
        C-->>C: Deterministic fallback report
    end
    C->>BC: Anchor audit on-chain
    BC-->>C: Transaction ID
    C-->>B: Final consolidated result
    B->>H: Save audit entry
    B-->>F: JSON response
    F-->>A: Render report + metrics + history
```

## 11. 🔐 AuditVault — Encrypted Paywall Flow

```mermaid
sequenceDiagram
    participant AGENT as 🤖 AI Agent
    participant VAULT as 🔐 AuditVault
    participant ALGO as ⛓️ Algorand
    participant AES as 🔒 AES-256

    AGENT->>VAULT: Request audit report
    VAULT->>AGENT: Payment required (0.02 ALGO)
    AGENT->>ALGO: Send micro-payment (X402 protocol)
    ALGO-->>VAULT: Payment confirmed
    VAULT->>AES: Decrypt report
    AES-->>VAULT: Plaintext report
    VAULT-->>AGENT: Deliver encrypted audit
    VAULT->>VAULT: Log revenue in agentic dashboard
```
